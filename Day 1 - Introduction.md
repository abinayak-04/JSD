# Java 
* In 1991 Java was first introduced by James Gosling at sun micro system.
* Java initial starting Name was OAK Language.
* In 1995 OAK is renamed to Java.
* Java was sold to oracle corporation.
* From version jdk 1.8 onwards java is maintained by oracle.
* The cost of java sold was 2.5 billion dollar.
* Java is an platform independent languages (support all).
* Java is case sensitive Language.

Example:
-> classname: first letter should be caps
-> var, method: first letter should be small

* Java extension is .java
* Java is a both compiler and interpreter.
* Java is an open source software.
* Compiler: Whole program error checking.
* Interpreter: line by line error checking.
* Compile Command for java program is javac filename.java
* To run the java program, the command is java classname
* Java default datatype: string.
* Java joining operator/concatenation operator : + 
* It supports OOPS partially.
* Java official website : www.oracle.com
* Java is a strongly type checked language (Compulsary to mention datatype in program)

Example: 
-> int V = 100;
-> float V2 = 33.34;
-> char C = '#';
-> string name = "Abi";

To check version or whether java is installed, the command is java --version
